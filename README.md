Pet-project

React + Node.js + Postges + Express.js

Создан для тренировки и развития навыков в указанном стэке. Временно заморожен. Вернусь к проекту после улучшения навыков верстки. 
На данный момент реализованы простые CRUD операции и авторизация (JWT-токены хранятся в LocalStorage, возможно позже будет переписано на cookies). Верстка хромает, будет переписана в ближайшем будущем. 

В планах реализация ленты новостей, планировщика задач, ролевой системы и, возможно, мини-игры.
