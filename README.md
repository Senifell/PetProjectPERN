Pet-project

React + Node.js + Postgres + Express.js

Создан для тренировки и развития навыков в указанном стэке.
На данный момент реализованы простые CRUD операции, авторизация (JWT-токены хранятся в LocalStorage, возможно позже будет переписано на cookies) и несколько интерфейсов со списками игр (кастомные игры + игры из Steam (API Steam). Верстка хромает, будет переписана в ближайшем будущем. 

Проект планируется как рулетка компьютерных игр, то есть пользователь может составить коллекцию игр и с помощью рулетки выбрать во что поиграть.
